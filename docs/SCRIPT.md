# Scripts

```bash
# 启动 Vite 开发服务器
npm run dev
```

```bash
# 执行 TypeScript 类型检查 + 构建生产版本
npm run build
```

```bash
# 构建后通过 Wrangler 本地预览
npm run preview
```

```bash
# 仅构建生产版本（跳过类型检查）
npm run build-only
```

```bash
# 运行 TypeScript 类型检查
npm run type-check
```

```bash
# 构建并部署到 Cloudflare
npm run deploy
```

```bash
# 根据 wrangler.jsonc 生成 Cloudflare Worker 类型定义
npm run cf-typegen
```

```bash
# ESLint 检查代码质量
npm run lint
```

```bash
# ESLint 检查并自动修复
npm run lint:fix
```

```bash
# Prettier 检查代码格式
npm run format
```

```bash
# Prettier 自动格式化所有文件
npm run format:fix
```
